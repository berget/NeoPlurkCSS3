上一版的CSS，因為過了一年半，經過多次Plurk官方改版，有些語法已經不相容出問題了，因此重新改版

NeoPlurkCSS3 佈景主題，延續舊版的主題，進行程式碼全面優化、縮減，並修正數個BUG，在搭配過去一年半眾多使用者提供的建議與想法，調整/新增版面配置

**NeoPlurkCSS3 佈景採用CSS3技術，IE瀏覽器可能會有部分不相容**
